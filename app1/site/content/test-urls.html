Check GetLegendGraphic STYLEs

Gradient is either predefined or can be specified in STYLE(S) param as ordered tuples of (Stop,R,G,B,A) each as a float in range 0..1. "Stop" is where this color should stop within gradient. "A" is transparency e.g. 0,0.8,0,0,1,0.5,1,1,1,1,0.9,1,1,1,1 (3 tuples: (0,0.8,0,0,1), (0.5,1,1,1,1) and (0.9,1,1,1,1))

/wms/?SERVICE=WMS&REQUEST=GetLegendGraphic&STYLE=heat/seth/red-yellow-green/5/0.4
/wms/?SERVICE=WMS&REQUEST=GetLegendGraphic&STYLE=heat/seth/red-yellow-trans2/5/0.4
/wms/?SERVICE=WMS&REQUEST=GetLegendGraphic&LAYER=brandweer&STYLE=heat/seth/0,0.8,0,0,1,0.5,1,1,1,1,0.9,1,1,1,1/5/0.4
/wms/?SERVICE=WMS&REQUEST=GetLegendGraphic&LAYER=brandweer&STYLE=heat/seth/0,0,0,1,1,0.5,0,1,0,1,0.9,1,1,1,1/5/0.4
