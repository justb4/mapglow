About MapGlow

MapGlow enables displaying your geodata as heatmaps. What sets MapGlow apart from other heatmap tools:

  • works with common vector data sources like PostGIS and ESRI Shapefiles
  • integrates with standard tools like UMN MapServer
  • serves heatmaps via the standardized Web Map Service (WMS)
  • allows for client-driven heatmap styling

MapGlow can thus be utilized within most GIS mapping applications. In addition, MapGlow's algorithmic extensibility allows for fine-grained tailoring for specific heatmap contexts.

How MapGlow works

MapGlow is basically a UMN MapServer interceptor written in Python MapScript. Heatmap generation is triggered by values of the STYLE parameter within WMS GetMap requests. When a heatmap is requested, MapGlow will query features (points) from the related MapServer Layer and draw a heatmap using the client-supplied parameters like gradients and algorithm-specific values.
